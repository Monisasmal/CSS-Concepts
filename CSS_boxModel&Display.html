Q31 ‚Äî Explain the CSS Box Model in detail.

  • Content ‚Üí text, images inside the element.
  • Padding ‚Üí space between content & border.
  • Border ‚Üí boundary around padding.
  • Margin ‚Üí space outside the border (separates from neighbors).

  width: 200px;
  padding: 10px;
  border: 5px solid black;
  margin: 20px;
}

Q33 ‚Äî What are replaced elements in CSS?

Replaced elements are those whose content is not controlled by CSS but by external resources (browser decides rendering).
Examples: <img>, <video>, <input>.

‚û° display: none ‚Üí removes element from DOM flow (no space).
‚û° visibility: hidden ‚Üí hides element but still takes up space.

‚ñ∂ em ‚Üí relative to parent font size.

‚û° Float (old method): used for layouts but difficult (clearfix hacks).
‚û° Flexbox/Grid (modern): flexible, responsive, no clearfix needed.

Q39 ‚Äî Difference between inline vs inline-block vs block with example?

div { display: block; }        /* full width */
